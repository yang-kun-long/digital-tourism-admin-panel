# 用户内容管理功能说明

## 功能概述

用户内容管理模块用于管理和审核用户上传的模板和照片。该功能支持三种管理员角色访问：
- `super_admin` - 超级管理员
- `content_admin` - 内容管理员
- `support_admin` - 支持管理员

## 功能模块

### 1. 用户内容概览 (`/user-content`)

**位置**: 侧边栏 > 用户内容 > 内容概览

**功能**:
- 显示统计卡片：用户模板总数、用户照片总数、待审核模板数、待审核照片数
- 快捷入口：快速跳转到用户模板管理和用户照片管理
- 最近待审核内容：显示最近 10 条待审核的模板和照片
- 支持快速审核（通过/拒绝）

### 2. 用户模板管理 (`/user-content/templates`)

**位置**: 侧边栏 > 用户内容 > 用户模板

**功能**:
- 查看所有用户上传的模板
- 筛选条件：
  - 分类（景区主题、风格分类、场景打卡）
  - 审核状态（待审核、已通过、已拒绝）
  - 关键词搜索（模板名称或创建者）
- 操作：
  - 查看详情：查看模板的完整信息
  - 通过审核：将模板状态改为"已通过"
  - 拒绝审核：将模板状态改为"已拒绝"，需填写拒绝原因
  - 删除：永久删除模板
- 分页显示

### 3. 用户照片管理 (`/user-content/photos`)

**位置**: 侧边栏 > 用户内容 > 用户照片

**功能**:
- 查看所有用户上传的照片
- 筛选条件：
  - 所属模板
  - 审核状态（待审核、已通过、已拒绝）
  - 照片类型（用户照片、官方照片）
  - 关键词搜索（照片名称或上传者）
- 操作：
  - 查看详情：查看照片的完整信息
  - 通过审核：将照片状态改为"已通过"
  - 拒绝审核：将照片状态改为"已拒绝"，需填写拒绝原因
  - 删除：永久删除照片
- 分页显示
- 显示内容审核状态（已通过/审核中/未通过）

## 使用流程

### 审核用户模板

1. 进入"用户内容 > 用户模板"页面
2. 可以通过筛选条件找到待审核的模板（状态选择"待审核"）
3. 点击"查看"按钮查看模板详情
4. 根据模板内容决定：
   - 点击"通过"按钮：模板将显示在用户端
   - 点击"拒绝"按钮：填写拒绝原因，用户可以看到拒绝原因

### 审核用户照片

1. 进入"用户内容 > 用户照片"页面
2. 筛选待审核的照片（状态选择"待审核"）
3. 点击"查看"按钮查看照片详情
4. 根据内容决定通过或拒绝
5. 注意查看"内容审核"状态，这是系统自动审核的结果

### 快速审核

在"用户内容 > 内容概览"页面，可以快速查看最近待审核的内容并进行审核操作，无需进入详细列表页面。

## 注意事项

1. **删除操作不可恢复**：删除模板或照片后无法恢复，请谨慎操作
2. **拒绝需填写原因**：拒绝审核时必须填写拒绝原因，帮助用户了解问题
3. **图片加载**：云存储图片会自动转换为临时 URL，有效期 2 小时
4. **权限控制**：所有三种管理员角色都可以访问用户内容管理功能
5. **内容审核**：照片有系统自动内容审核，但最终审核由管理员决定

## 后端要求

该功能需要以下 CloudBase 云函数支持，详见 `CLOUDBASE_FUNCTIONS.md`：

- `getUserTemplates` - 获取用户模板列表
- `getUserPhotos` - 获取用户照片列表
- `reviewUserTemplate` - 审核用户模板
- `reviewUserPhoto` - 审核用户照片
- `deleteUserTemplate` - 删除用户模板
- `deleteUserPhoto` - 删除用户照片
- `getUserContentStats` - 获取统计数据
- `getAllTemplates` - 获取所有模板（用于筛选）

## 开发说明

### 文件结构

```
src/views/user-content/
├── UserContentDashboard.vue   # 用户内容概览页面
├── UserTemplateList.vue        # 用户模板列表页面
└── UserPhotoList.vue           # 用户照片列表页面
```

### 路由配置

路由已在 `src/router/index.ts` 中配置：
- `/user-content` - 用户内容概览
- `/user-content/templates` - 用户模板管理
- `/user-content/photos` - 用户照片管理

### 菜单配置

菜单已在 `src/config/menu.ts` 中配置，显示在侧边栏"用户内容"分组下。

### 图片处理

使用 `@/utils/cloudbase.ts` 中的工具函数：
- `processImageUrl(url)` - 处理单个图片 URL
- `batchProcessImageUrls(urls)` - 批量处理图片 URL
- 自动缓存临时 URL，减少 API 调用

## 数据库结构

### templates 集合

用户模板存储在 `templates` 集合中，通过 `isOfficial: false` 区分用户模板和官方模板。

### photos 集合

用户照片存储在 `photos` 集合中，包含以下关键字段：
- `photoUrl` - 照片 URL
- `templateId` - 所属模板 ID
- `isOfficial` - 是否官方照片
- `status` - 审核状态（pending/approved/rejected）
- `contentCheckStatus` 或 `checkStatus` - 内容审核状态
- `uploaderName` 或 `userName` - 上传者名称
