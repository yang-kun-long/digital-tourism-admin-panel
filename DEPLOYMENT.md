### 步骤 1：部署云函数

1. 打开微信开发者工具
2. 在项目中找到 `cloudfunctions/adminLogin` 文件夹
3. 右键点击 `adminLogin` 文件夹
4. 选择"上传并部署：云端安装依赖"
5. 等待部署完成

### 步骤 2：配置云函数权限

云函数需要访问 `admins` 集合，确保：
1. 云函数有读取 `admins` 集合的权限
2. 云函数有更新 `admins` 集合的权限（用于更新最后登录时间）

### 步骤 3：测试管理后台

1. 进入 `admin-panel` 目录
2. 确保开发服务器正在运行：

3. 访问 http://localhost:5173
4. 使用以下凭据登录：

   - 密码：`admin`

## 功能说明

### 登录流程

1. 用户在登录页面输入用户名和密码
2. 前端调用 `adminLogin` 云函数
3. 云函数验证用户名和密码（MD5 哈希）
4. 验证成功后返回管理员信息
5. 前端保存管理员信息到 localStorage
6. 跳转到模板审批页面

### 权限说明

- **super_admin（超级管理员）**：
  - 可以审批模板
  - 可以删除模板
  - 未来可以创建新的管理员账户

- **admin（普通管理员）**：
  - 可以审批模板
  - 可以删除模板

### 安全特性

1. **密码加密**：使用 MD5 哈希存储密码
2. **账户状态检查**：禁用的账户无法登录
3. **登录状态持久化**：使用 localStorage 保存登录状态
4. **自动登录**：刷新页面后自动恢复登录状态

## 数据库结构

### admins 集合

```javascript
{
  _id: string,           // 管理员ID
  username: string,      // 用户名（唯一）
  password: string,      // 密码（MD5 哈希）
  role: string,          // 角色（super_admin/admin）
  status: string,        // 状态（active/disabled）
  createdAt: number,     // 创建时间
  lastLoginTime: number  // 最后登录时间
}
```

### 索引

- `username`：唯一索引，确保用户名不重复

## 故障排查

### 问题 1：云函数调用失败

**症状**：登录时提示"登录失败，请稍后重试"

**解决方案**：
1. 检查云函数是否已正确部署
2. 在微信开发者工具的云开发控制台查看云函数日志
3. 确认云函数名称为 `adminLogin`

### 问题 2：用户名或密码错误

**症状**：输入正确的用户名密码仍然提示错误

**解决方案**：
1. 检查 `admins` 集合中是否存在该账户
2. 确认密码是否为 MD5 哈希值
3. 默认账户：username=`admin`, password=`21232f297a57a5a743894a0e4a801fc3`

### 问题 3：登录后无法加载模板

**症状**：登录成功但模板列表为空或报错

**解决方案**：
1. 检查 CloudBase 匿名登录是否已启用
2. 检查 `templates` 集合的权限设置
3. 查看浏览器控制台的错误信息
